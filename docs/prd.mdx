# kaikosen
## 目的
<aside>
💡高専人の「あれ…この人前あったよな…？」をなくし、交流ハードルを下げる
</aside>


## 機能
- イベント参加時、過去イベントで同席した人を表示し、プロフ見たり、チャットとか送れるようにする。
    - お互いに繋がると、ノードが接続される。とか。

## MVP
- /(top)ページで、センターに自分がリンクしたことのある高専生をリンク構造で動的に表示(Obsidianのリンク機能をイメージして)、アイコンをタップしたらその人の詳細を見れるようにする。
- /accountページで、自分の名前(ニックネーム)、高専名、スキル･嗜好(#タグ式)を設定できるようにする。初回ログインの際はこのページか専用のページでアカウントを先に登録させる。
- イベントに参加したもの同士で、/cardを開いてQRコードの読み取りで繋がり交換が行われる
- 繋がる時に、その人の二つ名を設定する。
- /calendarに高専関連のイベント告知を載せる。/calendar/[id]にはそれぞれのイベントの詳細、メインURLに飛ぶためのボタン、ユーザーが参加表明ができるボタンを配置。
  - 繋がった人がイベントに参加するとき、通知を送る。
